# Yorkshire Transport & Storage Employment Analysis (2015-2024)

This project explores employment trends in Yorkshireâ€™s **transport and storage sector** using a set of interconnected data visualisations created in **R**. By combining multiple chart types, the analysis highlights **regional differences**, **structural changes**, and **long-term patterns** between 2015 and 2024.

The visualisations are designed to support **clear interpretation**, **accessibility**, and **insightful analysis** of how labour distribution has evolved across the region.

### 1. Parallel Coordinates Plot
Compares multiple variables across different regions of Yorkshire to identify structural differences.  

![Parallel Coordinates Plot](visualisation/parallel_coordinates.png)

### 2. Stacked Area Chart
Shows cumulative growth and the proportion of different sub-sectors over the 2015-2024 period.  

![Stacked Area Chart](visualisation/stacked_area_chart.png)

### 3. Alluvial Diagram
Visualizes the flow and structural changes in labor distribution between categories over time.  

![Alluvial Diagram](visualisation/alluvial_diagram.png)

### 4. Radial Heatmap
A circular representation showing the intensity of employment trends and seasonal/regional density.  

![Radial Heatmap](visualisation/radial_heatmap.png)

##  Tools & Libraries
This analysis was performed using **R Statistical Software** with the following libraries:
* **tidyverse** - Data manipulation & visualization
* **janitor** - Data cleaning
* **GGally** - Parallel Coordinates Plot
* **viridis** - Accessible color palettes
