# Yorkshire Transport & Storage Employment Analysis (2015-2024)
This project explores employment trends in Yorkshire‚Äôs transport and storage sector using a set of interconnected data visualisations created in R. By combining multiple chart types, the analysis highlights regional differences, structural changes, and long-term patterns between 2015 and 2024. 

The visualisations are designed to support clear interpretation, accessibility, and meaningful insight into how labour distribution has evolved across the region.
### 1. Parallel Coordinates Plot
Used to compare multiple variables across different regions of Yorkshire to identify structural differences.
![Parallel Coordinates Plot](visualisation/parallel_coordinates.png.png)

### 2. Stacked Area Chart
Visualizes the cumulative growth and the proportion of different sub-sectors over the 2015-2024 period.
![Stacked Area Chart](visualisation/stacked_area_chart.png.png)

### 3. Alluvial Diagram
Shows the flow and structural changes in labor distribution between different categories over time.
![Alluvial Diagram](visualisation/alluvial_diagram.png.png)

### 4. Radial Heatmap
A circular representation showing the intensity of employment trends and seasonal/regional density.
![Radial Heatmap](visualisation/radial_heatmap.png.png)
